# FaceFitAR 🎭📱

FaceFitAR is an Android application that uses **CameraX** for real-time face detection and applies AR filters on the face.

## 🚀 Features
- Real-time face detection  
- AR filters overlay on face  
- CameraX integration  
- Firebase Authentication (Login/Signup)  
- Smooth and responsive UI  

## 🛠️ Tech Stack
- Kotlin 
- Android Studio  
- CameraX  
- Firebase (Auth & Firestore)
